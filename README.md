# Noor I. - Professional Portfolio

Welcome to the open-source repository for my personal portfolio website!

I am Noor I., a Business Informatics student with a strong focus on digital innovation, Search Engine Optimization (SEO), and content creation. This portfolio serves as a digital resume and a showcase of my latest blogs, articles, and thought leadership at the intersection of business, technology, and data.

## Features

- **Modern & Responsive UI**: Designed with a sleek, glassmorphic dark theme.
- **Dynamic Animations**: Fluid scrolling, hover effects, and interactive elements.
- **SEO Optimized**: Built for performance, accessibility, and high search engine visibility.
- **Content Showcase**: Dedicated sections for recent articles, blogs, and professional experience.

## Technologies Used

- [Next.js](https://nextjs.org/) (App Router)
- [React 19](https://react.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)
- [Lucide Icons](https://lucide.dev/)

## Getting Started

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/ranaahsana16-gif/portfoilio.git
   ```

2. Navigate into the directory:
   ```bash
   cd portfoilio
   ```

3. Install dependencies:
   ```bash
   pnpm install
   ```

4. Run the development server:
   ```bash
   pnpm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

This portfolio is configured to be instantly deployed on [Vercel](https://vercel.com/). Simply connect this repository to a new Vercel project, and it will automatically build and deploy.
